Cheat sheet de tags de HTML

a - Hyperlinks

Sirven para hacer un link de una página a otro recurso, puede ser otro archivo en el servidor en donde corre el archivo html, u otra página web.

Por ejemplo: Visita éste enlace impresionante

    
    <a href="https://xxxxxxx.com">Visita nuestro sitio web</a>


p - Párrafo

Define un párrafo de texto.

Por ejemplo: Este es un ejemplo de un párrafo en HTML.

    
    <p>Este es un ejemplo de un párrafo en HTML.</p>


h1, h2, ..., h6 - Encabezados

Se usan para definir encabezados de HTML, donde h1 es el más importante y grande y h6 el menos importante.

Por ejemplo:

Encabezado Principal h1

Encabezado Secundario h2

    
    <h1>Encabezado Principal h1</h1>
    <h2>Encabezado Secundario h2</h2>


div - Contenedor para dividir

Se usa para agrupar bloques de elementos en un contenedor, y luego darles diferentes usos como estilizarlos o seleccionarlos con JS

Por ejemplo:

Este es un bloque div

    
    <div>Este es un bloque div</div>


span - Contenedor en línea

Se usa para agrupar elementos en línea, lo que no ocupa toda la línea y muchas veces es más eficiente para aplicar estilos

Por ejemplo: Este texto está dentro de un span.

    
    <span>Este texto está dentro de un span.</span>


strong - Texto Fuerte

Marca el texto como importante y lo muestra en negritas.

Por ejemplo: Este texto es importante.

    
    <strong>Este texto es importante.</strong>


i - Texto en Itálica

Se usa para indicar que un texto debe ser mostrado en itálica.

Por ejemplo: Este texto está en itálica.

    
    <i>Este texto está en itálica.</i>


u - Texto Subrayado

Se usa para subrayar un texto.

Por ejemplo: Este texto está subrayado.

    
    <u>Este texto está subrayado.</u>


b - Texto en Negritas

Hace que el texto se muestre en negritas.

Por ejemplo: Este texto está en negritas.

    
    <b>Este texto está en negritas.</b>


strike - Texto Tachado

Se usa para mostrar el texto como tachado.

Por ejemplo: Este texto está tachado.

    
    <strike>Este texto está tachado.</strike>


small - Texto Pequeño

Hace que el texto se muestre más pequeño de lo normal comparado con un texto en un elemento <p></p>

Por ejemplo: Este texto es pequeño.

    
    <small>Este texto es pequeño.</small>


em - Énfasis

Enfatiza el texto, generalmente mostrándolo en itálica.

Por ejemplo: Este texto está enfatizado.

    
    <em>Este texto está enfatizado.</em>


header - Cabecera

Representa un contenedor para contenido introductorio o un conjunto de enlaces de navegación.

Por ejemplo:

Esta es la cabecera de la página

    
    <header>Esta es la cabecera de la página</header>


section - Sección

Define una sección en un documento, es como el div pero los SE lo detectan y da mejores puntuaciones en el SEO

Por ejemplo:

Esta es una sección

    
    <section>Esta es una sección</section>


article - Artículo

Se usa para definir contenido independiente y autocontenido, e puede estilizar independientemente éste elemento

Por ejemplo:

Este es un artículo

    
    <article>Este es un artículo</article>


aside - Barra Lateral

Se usa para contenido que está relacionado con lo principal de la página, como puede ser el glosario de un párrafo, contenido adicional, etc, igual principalmente sirve para el SEO, ya que funcionalmente es igual que un elemento de párrafo.

Por ejemplo:

Esta es una barra lateral

    
    <aside>Esta es una barra lateral</aside>


footer - Pie de Página

Define el pie de página de un documento o sección.

Por ejemplo:

Este es el pie de página

    
    <footer>Este es el pie de página</footer>


nav - Navegación

Se usa para definir un conjunto de enlaces de navegación para navegar dentro de la página principalmente, o también se puede navegar hacia enlaces externos pero no es muy usando dentro de un <nav></nav>

Por ejemplo:

Este es el menú de navegación

    
    <nav>Este es el menú de navegación</nav>


img - Imagen

Se usa para insertar imágenes en un documento HTML.

Por ejemplo: Perdiste!! 

    
    <img src="imagen.jpg" alt="Descripción de la imagen">


br - Salto de Línea

Inserta un salto de línea en un texto, o también entre elementos

Por ejemplo: Esto es un texto.
Después del salto de línea.

    
    Esto es un texto.<br>Después del salto de línea.


hr - Línea Horizontal

Se usa para separar contenido con una línea horizontal.

Por ejemplo: 

    
    <hr>


form - Formulario

Se usa para crear un formulario para captar información del usuario y mandarla ya sea a una API con un método POST o a una función de JS dentro del browser.

Por ejemplo:

Formulario de ejemplo 

    
    <form>
    <input type="text" placeholder="Field 1">
    <input type="text" placeholder="Field 2">
    <input type="text" placeholder="Field 3">
    <input type="submit" value="Submit">
    </form>


table - Tabla

Se usa para crear tablas de datos.

Por ejemplo:

Elemento 1 en row 1 Elemento 2 en row 1
Elemento 1 en row 2 Elemento 2 en row 2 
    
    <table>
    <tr>
    <td>Data 1</td>
    <td>Data 2</td>
    </tr>
    <tr>
    <td>Data 3</td>
    <td>Data 4</td>
    </tr>
    </table>


ul - Lista No Ordenada

Se usa para crear listas de ítems sin un orden específico, con bullet points

Por ejemplo:

  • Item de lista
  • Item de lista
    
    <ul>
    <li>Item de lista</li>
    <li>Item de lista</li>
    </ul>


ol - Lista Ordenada

Se usa para crear listas de ítems con un orden específico.

Por ejemplo:

  1. Primer item de la lista
  2. Segundo item de la lista
    
    <ol>
    <li>Primer item de la lista</li>
    <li>Segundo item de la lista</li>
    </ol>
